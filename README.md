# OwO.Whats-Th.is?
A Pomf-compatible Node.js image sharing platform running on AWS' infrastructure.
Uses S3 and DynamoDB in order to reduce costs and provide a highly-available
service.

# Contributing

2. `npm install`
3. Make your changes
4. Commit, push, and submit PR
